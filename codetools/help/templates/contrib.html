Quer contribuir ?

Está pensando em como contribuir? Exitem varias formas de contribuir com codigo opensource ou com a comunidade!

para contribuir nesse projeto você pode:

Programar!

Faça um fork do projeto no github, clone para a sua maquina, faça a alteração e envie um pull request! não sabe como?
olha esse video: Como criar seu primeiro pull request no GitHub

Testar

Você pode contribuir usando a ferramenta, testando, vendo o que falta e como melhorar, pode abrir issues no github para que outras pessoas contribuam.

Documentar

Vocês sabia que muitos projetos precisam de galera que saiba documentar software? ajudar a documentar uma funçaõ, ajudar em alguma tradução é bem importante para a comunidade.
Parece bobo mas é estremamente necessario.

Divulgar

Que tal fazer uma palestra sobre uma ferramenta ? ou ir escutar uma palestra em algum evento ?

Apoiar mantenedores

Que tal doar 5 reais para algum mantenedor da ferratenta? sempre é bom uma ajuda funanceira tambem.
