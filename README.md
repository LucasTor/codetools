# Code Tools
Repositorio do site [codetools](https://codetools.com.br/)

## Utilizando o makefile

### Para preparar o ambiente de desenvolvimento

```shell
make
```

### Para rodar o projeto

```shell
make run
```

### Para rodar as migrations

```shell
make migrate
```
